What is Java club?

We are an Application Development Club

From a basic standpoint, we develop applications using the Java coding language. However, this is not all that we do. We also have different sections of our club that are dedicated to specific tasks correlating to each of our projects. For instance, the Development team writes the actual code and debugs it, etc,. The Marketing team finds ways to advertise the apps that we create so that it reaches a large audience. In conclusion, Java Club is mainly an application development club, but we focus on more than just one aspect of development.

What We Hope to Accomplish

The main goals that we hope to accomplish as an application development club are as follows:

  • Gather members that have a genuine interest to develop their skill set and create meaninful programs
  • Help each member of our club to learn Java, even if it's only the basic principles
  • Develop an application that fulfills a need and is able to be used by any single user. Quality and simplicity are not seperate entities, they are one and the same.
  • Gain experience that will benefit each member in whatever career they choose to pursue.
